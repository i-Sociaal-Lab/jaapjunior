**IV074**

*Wat is het verschil tussen de verschillende uitvoeringsvarianten?*

De iStandaarden ondersteunen een aantal verschillende uitvoeringsvarianten: inspanningsgericht, outputgericht en taakgericht. De specifieke eigenschappen van deze uitvoeringsvarianten zijn beschreven in het document Handreiking uitvoeringsvarianten, deze is beschikbaar via www.istandaarden.nl/ibieb/handreiking-uitvoeringsvarianten-iwmo-en-ijw.
