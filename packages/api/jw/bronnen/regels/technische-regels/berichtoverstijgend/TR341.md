# TR341 Technische regel

## Definitie

Eenheid in Prestatie moet passen bij Eenheid in het ToegewezenProduct.

---

| Type | Technische regel |
| Controleniveau | berichtoverstijgend |
| Retourcode | 9341 |
| Bron | iJw standaard |
| Bestandstype | Technische regel |

---

## Toelichting

Dit betekent dat de eenheid in de declaratie gelijk moet zijn aan de eenheid uit de toewijzing. Bij toewijzing in uren mag ook in minuten gedeclareerd worden. Voor aspecifieke of generieke toewijzingen geldt dat de eenheid overeen dient te komen met de eenheid in de gemaakte contractafspraken.
