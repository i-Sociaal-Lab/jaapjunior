# TR374 Technische regel

## Definitie

Een verzoek mag alleen worden verstuurd als er geen onderhanden verzoek is voor de betreffende client van dezelfde aanbieder.

## Toelichting

Een verzoek is in behandeling totdat er ofwel een antwoordbericht met VerzoekAntwoord "Verzoek afgewezen" ofwel een toewijzingsbericht met dezelfde ReferentieAanbieder gestuurd is.
