# OP155 - Een (tijdelijke) beeindiging van levering, bevat een verwijzing naar de laatste melding van de start van de levering.

## Kernregel

Een (tijdelijke) beëindiging van levering bevat een verwijzing naar de laatste melding van de start van de levering.

Van toepassing op:

## Regeltekst

Een (tijdelijke) beëindiging van levering bevat een verwijzing naar de laatste melding van de start van de levering.

---

## Betekenis

Binnen JW307 moet een melding van beëindiging of tijdelijke beëindiging van levering gekoppeld zijn aan de meest recente melding van start levering.

De beëindigingsmelding:

- verwijst naar de laatste startmelding;
- legt de relatie tussen start en beëindiging vast;
- ondersteunt consistente verwerking van leveringsperiodes.

Dit geldt ook voor:

- tijdelijke beëindigingen;
- tijdelijke stops van ondersteuning;
- beëindigingen met reden beëindiging `21`.

Hierdoor blijft de historie van de levering eenduidig herleidbaar.

---

## Validatie

Een bericht voldoet aan OP155 indien:

1. een beëindiging verwijst naar de laatste melding start levering;
2. de relatie tussen start en beëindiging herleidbaar is;
3. de verwijzing overeenkomt met de actuele leveringsperiode.

Een bericht is ongeldig indien:

- geen verwijzing naar de laatste startmelding aanwezig is;
- verwezen wordt naar een onjuiste of verouderde startmelding;
- de relatie tussen levering en beëindiging niet vastgesteld kan worden.
