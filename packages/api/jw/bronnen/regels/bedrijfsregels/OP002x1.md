# OP002x1 Gezagsdrager bekend

**Afkorting:** OP

**Categorie:** Bedrijfsregels

**Document type:** iJw bedrijfsregels

**Versie volledig:** iJw 3.2.0

**Domein:** Jeugdwet

**Type:** bedrijfsregel

## Berichttype
- JW301

**Bron:** https://informatiemodel.istandaarden.nl/informatiemodel/ijw/3.2/regels/bedrijfsregel/op002x1/

**Samenvatting:** Indien een aanbieder in een Verzoek om toewijzing (VOT) heeft aangegeven dat de gezagsdrager bekend is, hoeft in de daaropvolgende toewijzing de gezagsdrager niet meegegeven te worden.

**Uitleg:** - Indien een aanbieder in een Verzoek om Toewijzing (VOT) heeft aangegeven dat de gezagsdrager bekend is, hoeft in de daaropvolgende toewijzing de gezagsdrager niet meegegeven te worden.
