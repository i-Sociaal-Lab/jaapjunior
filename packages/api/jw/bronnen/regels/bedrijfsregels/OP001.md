# OP001 — Van een cliënt wordt vastgelegd wie de gezagsdrager is

## Definitie
Van een cliënt moet worden vastgelegd wie de gezagsdrager is.  
Indien het een cliënt betreft van 16 jaar of ouder, kan het gezag over de jeugdige ook bij de cliënt zelf liggen.

---

## Samenvatting
Deze bedrijfsregel verplicht het registreren van de gezagsdrager van een jeugdige cliënt.  
Bij cliënten vanaf 16 jaar mag de cliënt zelf de gezagsdrager zijn.
