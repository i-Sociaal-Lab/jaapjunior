# OP001 - Van een client wordt vastgelegd wie de gezagsdrager is.

**Samenvatting:** Van een cliënt wordt vastgelegd wie de gezagsdrager is. Indien het een client betreft die 16 jaar of ouder is kan het gezag over de jeugdige ook bij de client zelf liggen.

**Uitleg:** Van een cliënt wordt vastgelegd wie de gezagsdrager is. Indien het een client betreft die 16 jaar of ouder is kan het gezag over de jeugdige ook bij de client zelf liggen.
