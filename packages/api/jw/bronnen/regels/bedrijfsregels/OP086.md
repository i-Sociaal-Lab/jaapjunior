# OP086 — Inspanningsgerichte en outputgerichte ondersteuning wordt altijd toegewezen via een toewijzingsbericht

## Kernregel

Ondersteuningsproducten met een inspanningsgerichte of outputgerichte uitvoering worden altijd door de gemeente toegewezen via een toewijzingsbericht.

Van toepassing op:

- JW301

## Regeltekst

Ondersteuningsproducten met een inspannings- of outputgerichte uitvoering worden altijd door de gemeente toegewezen via een toewijzingsbericht.

---

## Betekenis

Wanneer een ondersteuningsproduct wordt uitgevoerd volgens een:

- inspanningsgerichte uitvoeringsvariant;
- outputgerichte uitvoeringsvariant;

moet de gemeente hiervoor altijd een formeel toewijzingsbericht versturen.

Het toewijzingsbericht:

- legt de toewijzing officieel vast;
- vormt de basis voor levering en administratieve verwerking;
- zorgt voor eenduidige communicatie tussen gemeente en aanbieder.

Ondersteuning binnen deze uitvoeringsvarianten mag niet zonder toewijzingsbericht worden gestart of verwerkt.

---

## Validatie

Een bericht voldoet aan OP086 indien:

1. inspanningsgerichte ondersteuning via een toewijzingsbericht wordt toegewezen;
2. outputgerichte ondersteuning via een toewijzingsbericht wordt toegewezen;
3. de toewijzing afkomstig is van de gemeente;
4. de uitvoeringsvariant correct is vastgelegd.

Een bericht is ongeldig indien:

- een inspanningsgerichte uitvoering zonder toewijzingsbericht wordt verwerkt;
- een outputgerichte uitvoering zonder toewijzingsbericht wordt verwerkt;
- de toewijzing niet formeel door de gemeente is vastgelegd.
