# OP072 — Beëindiging levering pas toegestaan na melding start ondersteuning

## Kernregel

Een aanbieder mag pas melden dat de levering beëindigd is nadat eerder een melding start ondersteuning is gedaan.

In het bericht moet gerefereerd worden aan de start van de levering.

Van toepassing op:

- JW307

## Regeltekst

Pas na een melding start ondersteuning mag een aanbieder melden dat de levering beëindigd is.

In het bericht moet gerefereerd worden aan de start van de levering.

---

## Betekenis

Een beëindiging van levering mag uitsluitend worden geregistreerd wanneer eerder een melding start ondersteuning heeft plaatsgevonden.

Bij een melding beëindiging levering:

- moet een eerdere startmelding bestaan;
- moet het bericht verwijzen naar de start van de levering;
- moet de relatie tussen start en beëindiging eenduidig herleidbaar zijn.

Hierdoor blijft de levenscyclus van de ondersteuning consistent binnen het berichtenverkeer.

---

## Validatie

Een bericht voldoet aan OP072 indien:

1. voorafgaand een melding start ondersteuning aanwezig is;
2. de beëindigingsmelding verwijst naar de start van de levering;
3. de levering administratief herleidbaar is naar de oorspronkelijke startmelding.

Een bericht is ongeldig indien:

- een levering beëindigd wordt zonder eerdere startmelding;
- geen referentie naar de start van de levering aanwezig is;
- de relatie tussen start en beëindiging niet vastgesteld kan worden.
