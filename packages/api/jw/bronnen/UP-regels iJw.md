# Uitgangspunten (UP-regels)

De volgende uitgangspunten vormen de basis voor het berichtenverkeer binnen de Jeugdwet (iJw 3.2.0):

---
ID: UP001
Titel: Ondersteuning Jw berichtenfamilie
Versie: iJw 3.2
Domein: Jeugdwet
Type: uitgangspunt
Bron-link: https://informatiemodel.istandaarden.nl/informatiemodel/ijw/3.2/regels/uitgangspunt/up001/
Tekst: De Jw berichtenfamilie ondersteunt het toewijzen, leveren en declareren van Jeugdhulp in het kader van de Jeugdwet, geleverd als zorg in natura (ZIN).
Tekst_raw: 
- De Jw berichtenfamilie ondersteunt het:
  - toewijzen
  - leveren
  - declareren 
  van Jeugdhulp in het kader van de Jeugdwet, geleverd als zorg in natura (ZIN)
