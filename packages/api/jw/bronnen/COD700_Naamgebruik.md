# COD700: Naamgebruik

**Definitie:**  
Aanduiding naamgebruik (gecodeerd).

- **Status:** Productie  
- **Auteur:** Zorginstituut NL  
- **Bron:** https://informatiemodel.istandaarden.nl/informatiemodel/ijw/3.2/codelijsten/cod700/  
- **AI-agent:** Jw-agent  
---

## Codelijst

| Code | Omschrijving |
|:----:|:-------------|
| 1 | Eigen naam |
| 2 | Naam echtgenoot of geregistreerd partner of alternatieve naam |
| 3 | Naam echtgenoot of geregistreerd partner gevolgd door eigen naam |
| 4 | Eigen naam gevolgd door naam echtgenoot of geregistreerd partner |
| 5 | Pseudoniem |
| 6 | Niet-natuurlijk persoon |
