## NUM061: Aanduiding woonadres
**definitie**
Aanduiding woonadres - De aanduiding die wordt gebruikt voor adressen die niet zijn voorzien van de gebruikelijke straatnaam en huisnummeraanduidingen.

| Code | Waarde |
|------|--------|
| AB | Aan boord |
| BY | Bij |
| TO | Tegenover |
| WW | Woonwagen |
