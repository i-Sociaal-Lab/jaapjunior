# TR316 Technische regel

## Definitie

Debetregels mogen niet in hetzelfde bericht gecrediteerd worden.

---

| Retourcode | 0001 |
| Bron | iJw standaard |
| Bestandstype | Technische regel |

---

## Toelichting

VorigReferentieNummer mag geen waarde bevatten gelijk aan ReferentieNummer van een Prestatie in hetzelfde bestand. Creditering van een debet prestatie in hetzelfde bericht is niet toegestaan.
