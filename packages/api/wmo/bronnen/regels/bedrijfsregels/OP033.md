# OP033 - Voor het doorgeven van wijzigingen en correcties op een eerder verzonden bericht, moet gebruik gemaakt worden van de systematiek van status aanlevering.

**Afkorting:** OP

**Categorie:** Bedrijfsregels

**Document type:** iWmo bedrijfsregels

**Versie volledig:** iWmo 3.2.0

**Domein:** WMO

**Type:** bedrijfsregel

## Berichttype
- WMO305
- WMO307

**Onderwerp:** wijzigen, corrigeren

**Bron:** https://informatiemodel.istandaarden.nl/informatiemodel/iwmo/3.2/regels/bedrijfsregel/op033/

**Samenvatting:** Voor het doorgeven van wijzigingen en correcties op een eerder verzonden bericht, moet gebruik gemaakt worden van de systematiek van status aanlevering.

**Uitleg:** 
 - Correcties op een melding start product zijn uitsluitend toegestaan voor het geleverde product. Als de ondersteuning beeindigd is, kan de aanvang niet meer op deze manier gecorrigeerd worden. 
 een berichtklasse nieuw is (waarde 1) 
 een berichtklasse verwijderd moet worden (waarde 3). 
 Een verwijdering betekent dat de vorige aanlevering(en) met dezelfde sleutel als niet verzonden beschouwd moet worden.
