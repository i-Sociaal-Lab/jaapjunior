# OP071 - Elke relatie krijgt een uniek nummer per gemeente per client.
**Afkorting:** OP

**Categorie:** Bedrijfsregels

**Document type:** iWmo bedrijfsregels

**Versie volledig:** iWmo 3.2.0

**Domein:** WMO

**Type:** bedrijfsregel

## Berichttype
- WMO301

**Onderwerp:** relatienummer

**Bron:** https://informatiemodel.istandaarden.nl/informatiemodel/iwmo/3.2/regels/bedrijfsregel/op071/

**Samenvatting:** Elke relatie krijgt een uniek nummer per gemeente per client. Dit nummer mag niet gewijzigd worden.

**Uitleg:** 
 - Elke relatie krijgt een uniek nummer per gemeente per cliënt.  
 - Dit nummer mag niet gewijzigd worden.
