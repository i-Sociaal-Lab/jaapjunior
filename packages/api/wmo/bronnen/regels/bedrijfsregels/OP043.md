# OP043 - Indien aan een client een PGB is toegekend is, mag op hetzelfde moment voor hetzelfde product geen toewijzing zijn afgegeven.

**Afkorting:** OP

**Categorie:** Bedrijfsregels

**Document type:** iWmo bedrijfsregels

**Versie volledig:** iWmo 3.2.0

**Domein:** WMO

**Type:** bedrijfsregel

## Berichttype
- WMO301

**Onderwerp:** clientgegevens

**Bron:** https://informatiemodel.istandaarden.nl/informatiemodel/iwmo/3.2/regels/bedrijfsregel/op043/

**Samenvatting:** Als een cliënt een PGB heeft, mag voor hetzelfde product geen toewijzing zijn afgegeven. Een toewijzing eindigt uiterlijk op de dag vóór ingangsdatum PGB.

**Uitleg:** 
 - Als een cliënt een PGB heeft, mag voor hetzelfde product geen toewijzing zijn afgegeven.  
 - Een toewijzing eindigt uiterlijk op de dag vóór ingangsdatum PGB.
